= CouchRestAdapter

This project rocks and uses MIT-LICENSE.

== Requirements

* Rails > 4
* Couchrest
* Requires config/database.yml file


== Instalation

=== In Gemfile

<tt>gem 'couch_rest_adapter', git: 'https://github.com:amco/couch_rest_adapter.git'

=== Model Declaration

<tt>
  class User < CouchRestAdpater::Base
    use_default_database
  end
</tt>
