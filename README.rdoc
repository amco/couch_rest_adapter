= CouchRestAdapter

This project rocks and uses MIT-LICENSE.

== Requirements

* Rails > 4
* Couchrest
* Requires config/database.yml file


== Instalation

=== In Gemfile

    gem 'couch_rest_adapter', git: 'git://github.com/amco/couch_rest_adapter.git'

=== Model Declaration


  class User < CouchRestAdpater::Base
    use_default_database
  end
